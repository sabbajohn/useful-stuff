## [1.1.0] - 2025-01-03

### Added
- Mac Storage Manager v2.0 with complete rewrite
- Interactive fzf-based interface for better user experience
- Non-interactive CLI mode for automation and scripting
- Advanced storage analysis with real-time reporting
- Smart file detection (large files, old files, duplicates)
- Comprehensive cleanup options for all development tools
- Storage snapshots for measuring cleanup effectiveness
- Enhanced safety features with confirmation prompts
- Professional CLI with help, version, and debug modes
- Detailed test suite for Mac Storage Manager
- Comprehensive documentation and usage guides

### Enhanced
- Storage management utilities completely rewritten
- Better error handling and user feedback
- Color-coded output for improved readability
- Performance optimizations for large file scans
- Cross-platform compatibility improvements

### Mac Storage Manager Features
- User caches cleanup (~2-5GB typical savings)
- Xcode data management (5-50GB potential savings)  
- Docker cleanup (~10-100GB potential savings)
- Node.js cleanup with npkill integration (~1-20GB typical savings)
- Package managers cleanup (Homebrew, npm, Yarn, pip)
- Git repositories optimization (10-50% size reduction)
- iOS Simulators cleanup and management
- System logs cleanup with permission handling
- Trash management with safety confirmations
- Duplicate files detection with hash verification

- Professional documentation and usage examples
